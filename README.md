# Html-Starter
Html skeleton for common use

# Background
1.install node.js
2.node -v , npm -v (update node: n latest or n stable)
3.npm install cnpm -g --registry=https://registry.npm.taobao.org => cnpm -v 


# Usage
1.cnpm install
2.gulp clean
3.gulp build
4.gulp server
5.gulp (gulp clean + gulp build + gulp server)
